---
title: "Run Agent"
api: "POST /v1/agent"
description: "Send a message with optional capabilities and files. Get finished work back."
---

<RequestExample>
```bash cURL
curl -X POST https://api.stewrd.dev/v1/agent \
  -H "Authorization: Bearer sk-stw_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Research the top 5 CRM tools and create a comparison spreadsheet",
    "capabilities": ["research", "documents"]
  }'
```

```typescript TypeScript
const response = await fetch('https://api.stewrd.dev/v1/agent', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer sk-stw_your_key',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    message: 'Research the top 5 CRM tools and create a comparison spreadsheet',
    capabilities: ['research', 'documents'],
  }),
})

const data = await response.json()
```

```python Python
import requests

response = requests.post(
    'https://api.stewrd.dev/v1/agent',
    headers={
        'Authorization': 'Bearer sk-stw_your_key',
        'Content-Type': 'application/json',
    },
    json={
        'message': 'Research the top 5 CRM tools and create a comparison spreadsheet',
        'capabilities': ['research', 'documents'],
    },
)

data = response.json()
```
